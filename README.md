# Währung-Umrechner

## Motivation

Im Mathematik-Unterricht hatten wir Währungsumrechnungen als Hausaufgaben.  
Mein Ziel war es, einen einfach bedienbaren Währungsumrechner zu programmieren.

## Funktionen

- Umrechnung der Währungen **Franken (Fr)**, **Dollar ($)** und **Pesos (₱)**.
- Addition von Beträgen aus verschiedenen Währungen.
- Ausgabe des Resultats in einer frei wählbaren Zielwährung.
- Einfache Bedienung über eine grafische Benutzeroberfläche (GUI).



## Motivation

Im Mathematik-Unterricht hatten wir Währungsumrechnungen als Hausaufgaben.
Mein Ziel war ein einfach bedienbarer Währungsumrechner zu programmieren.
Funktionen: Die Währung Franken, Pesos und Dollar sollen umgerechnet werden können.



## Besonderheit

Das spezielle an meinem Umrechner ist, dass Beträge von verschiedenen Währungen zusammen addiert werden und das Resultat schlussendlich in Franken angegeben wird.



### Beispiel

Eingabe:

```text
112₱ + 336$
```

Ausgabe:

```text
304.19 Fr
```
